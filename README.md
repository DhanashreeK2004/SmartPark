SmartPark is a Smart Parking Management System built using Spring Boot and MySQL.

## Features
- User Registration & Login
- Automatic Parking Slot Allocation
- Slot Release System
- Real-time Available Slot Tracking
- REST API Backend

## Tech Stack
- Java
- Spring Boot
- Spring Security
- MySQL
- Maven

## How to Run
1. Clone the repository
2. Update application.properties with your MySQL credentials
3. Run the Spring Boot application
4. Access APIs using Postman or frontend
